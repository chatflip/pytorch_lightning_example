# pytorch_lightning_example

<https://github.com/chatflip/tutorial_pytorch_japanese/tree/develop> をpytorch-lightning移行

## Requirement

- Ubuntu
- [uv](https://docs.astral.sh/uv/)

## Installation

### 実行環境導入(仮想環境)

```bash
make install
```

## Usage

### formatter, linter, type checker

```bash
make format
make lint
```
